# German

## Typing

- umlaut chord (option U)
- esszett (option S, directly)

## Grammar

- four cases
  - nominativ
  - dativ
  - akkusativ
  - genitiv

## Pronouns

- ich
- du
- er, sie, es
- wir
- ihr
- sie (they), Sie (you formal)

## Articles

- der, die, das (definitive, masc fem neut)
- ein, eine, ein (indefinite)

## Nouns

- die Sprache / language
- das Spiel / game
  - Lieblingscomputerspiel
- die Wohnung / apartment
- deine Hobbys
- das Singen (singing)
- Fahrrad fahren (biking)
  - fahren = to drive
- das Rad = wheel
- das Auto = car
- das Essen = food
- die Pandemie
- Beruf / job
- Softwareentwickler = software developer
- die Katze (cat)
- der Hund (dog)
- die Tasse (cup)

## Adjectives

- Liebelings / favorite
  - attached to anything
  - Was ist deine Lieblingsprogrammiersprache
- deine (your)

## Verbs

- ich mag (i like)
  - Ich mag Volleyball
- Wo wohnst du? (where do you live)
  - Ich wohne in
- Arbeitest du in Queens? (do you work)
  - arbeiten
  - Ich arbeite zu Hause. (at home)

## Unsorted

- hallo / hello
- nie / niemals = never
- Wie alt bist du?
  - Ich bin sieben-und-dreißig Jahre alt.
- Wie heißt er
  - what's his name?
